# Pokemon Battle System - Implementación de QA

## 🚀 Reporte de Aseguramiento de Calidad
Este proyecto incluye una suite de pruebas unitarias enfocada en validar las reglas del negocio, logrando una cobertura del 74% del sistema total.

### Principales Logros
* **Reglas del Negocio:** Validación al 100% de las 4 reglas solicitadas mediante pruebas unitarias en `src/test/java`.
* **Motor de Batalla:** Implementación de simulación de entrada (`System.in`) para cubrir flujos interactivos de combate.
* **Calidad de Código:** Integración con JaCoCo para monitorear el progreso de cobertura mediante `mvn verify`.
